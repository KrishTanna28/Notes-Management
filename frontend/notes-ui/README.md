# Notes UI

React + Vite frontend for the Notes Management System.

```bash
npm install
npm run dev
```

On Windows PowerShell, use `npm.cmd install` and `npm.cmd run dev` if `npm.ps1`
is blocked by execution policy.

Start the backend from `backend/notes-api`, then open `http://localhost:5173`.
The UI supports note listing, create/edit forms, delete confirmation, full note
viewing, search, pinned notes, tags, loading, empty, and error states.
